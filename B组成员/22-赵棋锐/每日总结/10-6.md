## 数学模型的随机化处理与运算
```
class Network(object):
    # 随机化数学模型
    def __init__(self,num_of_weights):
        py.random.seed()
        self.w=py.random.randn(num_of_weights,1)
        print(self.w)
        W=self.w
        self.b=0.
    # 计算函数的预测值
    def forward(self,x):
        z=py.dot(x,self.w)+self.b
        return z
    # 计算函数的损失值
    def loss(self,z,y):
        error=z-y
        cost=error*error
        cost=py.mean(cost)
        return cost,W
```
## 数据处理，判断丢失度
