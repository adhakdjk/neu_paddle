## 数据集探索
### 分类数据
```
data.groupby(['blablba'])
```
把分类的数据称为***阶级数据
* 同意义不同阶级的数据求平均数(意义与阶级意义可以不同)
```
data.groupby(['_1'])['_2'].max()
data.groupby(['_1'])['_2'].min()
data.groupby(['_1'])['_2'].mean()
```
很香的是  
```
data.groupby(['_1'])['_2'].idmax()
```
就可以~人肉~了
